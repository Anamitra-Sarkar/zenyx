================================================================================
QUICK START
================================================================================

1. Launch TPU v5e-8:
   gcloud compute tpus create zenyx-1t --accelerator-type=v5e-8

2. SSH and setup:
   ssh user@zenyx-1t
   git clone https://github.com/zenyx-ai/zenyx.git && cd zenyx
   pip install -e .
   pip install -U 'jax[tpu]' flax optax datasets transformers

3. Set token:
   export HF_TOKEN="your_token_here"

4. Run training (8K context):
   python train/zenyx_tpu_v5e8_1t_1m.py --train --steps 10000 --max-seq-len 8192

5. Scale to 1M context:
   python train/zenyx_tpu_v5e8_1t_1m.py --train --steps 50000 --max-seq-len 1000000

================================================================================
MODEL SPECS
================================================================================

Parameters:        1 Trillion (1,000,000,000,000)
Layers:           128 unique blocks × 8 recurrences = 1,024 effective
Context:          1,000,000 tokens (1M) or 8,192 tokens (8K train start)
Vocabulary:       262,144 tokens
Attention:        Multi-head Latent Attention (MLA) — 100× KV compression
Position:         YaRN-scaled RoPE (extrapolate 8K → 1M)
Distributed:      Ring Attention (8 TPU cores)
Memory:           3-tier management (HBM → DRAM → NVMe)
Loss:             Chunked cross-entropy + multi-token prediction
Optimization:     AdamW + gradient clipping

================================================================================
HOW TO CUSTOMIZE
================================================================================

1. Replace data pipeline:
   Edit create_dummy_batch() function in train/zenyx_tpu_v5e8_1t_1m.py
   Replace with your dataset loading code

2. Change model size:
   Modify ZenyxConfig class:
   - num_layers: Change 128 for different sizes
   - hidden_size: Change 16_384 for different dimensions
   - num_heads: Change 128 for different attention heads

3. Adjust training hyperparameters:
   - learning_rate: 3e-4 (default)
   - weight_decay: 0.1 (default)
   - max_grad_norm: 1.0 (default)
   - batch_size: 256 (default)

4. Tune context length:
   - max_seq_len: 1_000_000 (default 1M)
   - Start with 8_192 for testing
   - Scale up gradually (8K → 16K → 32K → ... → 1M)
