================================================================================
TECHNICAL INNOVATIONS
================================================================================

1. RECURRENT LAYER MULTIPLYING (RLM)
   - 128 unique blocks × 1 recurrence = 128 effective layers
   - Enables 1 trillion parameters on TPU v5e Megapod
   - Memory efficient while maintaining depth

2. MULTI-HEAD LATENT ATTENTION (MLA)
   - Compresses KV to ~256 dimensions
   - 10-100x KV cache reduction
   - Maintains attention quality

3. YARN-SCALED ROPE
   - Frequency interpolation for long context
   - Train at 8K → Inference at 1M
   - No retraining required

4. CHUNKED CROSS-ENTROPY
   - Memory-efficient loss for large vocab
   - Process 32K vocab in 16 chunks
   - O(batch × seq × chunk) memory

================================================================================
COMMON ISSUES & SOLUTIONS
================================================================================

Issue: OutOfMemoryError on forward pass
  Fix: Enable activation checkpointing or reduce per-core batch

Issue: Slow training (< 100 tokens/sec)
  Fix: Profile with jax.profiler.trace(), check data I/O

Issue: NaN in loss
  Fix: Reduce learning rate, increase warmup steps

Issue: JAX import error
  Fix: Install jax[tpu] on TPU instance (not CPU)

Issue: Model initializes but forward fails on huge context
  Fix: Use pmap/pjit for distributed training across cores
